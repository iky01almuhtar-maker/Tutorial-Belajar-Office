Tutorial Microsoft Excel

Rumus, shortcut, dan tips untuk bekerja cepat di Excel.

Rumus dasar

  • SUM — Menjumlahkan nilai: =SUM(A1:A10)
  • AVERAGE — Rata-rata: =AVERAGE(A1:A10)
  • MAX / MIN — Nilai tertinggi/rendah

Rumus menengah

  • IF: =IF(A1>10,"Yes","No")
  • VLOOKUP: =VLOOKUP(key, table, col, FALSE)
  • HLOOKUP, COUNTIF, SUMIF

Tips & Shortcut

  • Ctrl + Shift + L = Filter
  • Ctrl + Arrow Key = Lompat data
  • F2 = Edit cell
  • Ctrl + ; = Masukkan tanggal

Tabel ringkasan rumus

RumusContohKegunaan
